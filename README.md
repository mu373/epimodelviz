# epimodelviz

Visualize epidemiological compartment models from YAML.

https://epimodelviz.vercel.app/

Reads basemodel YAML files following the [epymodelingsuite](https://github.com/mobs-lab/epymodelingsuite) schema.

## Setup

```bash
pnpm install
pnpm run dev
pnpm run dev --host 0.0.0.0 # optional: expose to network
```
